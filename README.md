Career Quest Bot (JARVIS)
**AI-Driven Guidance for Your Professional Journey**

Career Quest Bot (JARVIS) is an AI-powered career guidance chatbot designed to help students explore career opportunities, discover suitable career paths, and make informed professional decisions through an interactive conversational interface.

Project Overview

Career Quest Bot provides personalized career guidance by understanding user interests, skills, and career goals. It assists students with career exploration, educational pathways, resume guidance, interview preparation, and job search strategies, making career counseling more accessible and affordable.

Features

-  AI-Powered Career Guidance
-  Personalized Career Recommendations
-  Interactive AI Chatbot
-  Career Exploration
-  Resume Building Guidance
-  Interview Preparation Tips
-  Job Search Assistance
-  Skill Assessment
-  Educational Path Recommendations
-  Career Path Analysis

Objectives

- Provide personalized career guidance.
- Help students identify suitable career paths.
- Bridge the gap between education and industry requirements.
- Reduce dependency on expensive career counseling.
- Improve career decision-making using AI.

Installation

1. Clone the Repository

```bash
git clone https://github.com/harivasagar/career-quest-bot.git
```

2. Navigate to the Project

```bash
cd career-quest-bot
```

3. Start XAMPP

- Start Apache
- Start MySQL

4. Import Database

- Open phpMyAdmin
- Create a database
- Import the provided SQL file

5. Run the Project

Open your browser and visit:

```
http://localhost/Career-Quest-Bot/
```
